## Build Dockerfile
```
docker build -t [tag name] [buildcontext]
```
## Run container
```
docker run -v "$PWD"/C:/home --rm -ti [container name]
```
## how to use this Repo
```
$cd home // directory change

$gcc [filename] //compile

$./a.out
```
